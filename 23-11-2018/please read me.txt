INSTRUCTIONS/STEPS TO INSTALL THE SYSTEM ON THE MACHINE
=======================================================
1. Determine the server machine where the database will be installed
2. Then on machines you have to install the following application
	2.1 xammplite(application machine)
	2.2 mysql(database machine)
	2.3 sqlyog(database machine)
3. Then load the database on sqlyog
4. You have also to load the system to the folder called htdoc which is find on the xammplite folder
5. Then you can now stop the iis if you have it on your machine
6. Run xammplite application, you will see apach server on xammplite then you have to start it
7. Then to run the system you have to open chrome/internet explore then type http//localhost//carCompSystem//index.php on url
